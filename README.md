## ▶ Cara Menjalankan

```bash
git clone https://github.com/Sagasen/Web-Projects.git
cd Web-Projects/<nama-project>
```

Ikuti README di masing-masing folder project untuk setup & menjalankannya — tiap project punya kebutuhan environment yang berbeda.
